how to build you own hexo blog

hexo is a base for node.js frame ,you can use it build a static state web blog!

  • first you need a node.js
  • and you need a gitbash
  • you must register a github account

if you finished install node and git ,you should create a folder called blog or other something ,then find gitbush and open it.you gitbush maybe like this:

and find the name is blog folder use cd command
,when you in blog folder input command
‘’’ 

npm install -g hexo
//if this command is successful execution 
hexo init
//you successful install the hexo
hexo g
hexo s
//open http://localhost:4000 this is you blog local version

‘’’

how to write blog

  • find folder in hexo names ../hexo/source/_posts this folder is a markdown file storage,anything you create the new page will be storage in this folder
  • When you finish writing you must use “hexo g”command to generates a static page to the public directory.(when you first use this command you should execution “nmp install hexo-deployer-git –save”)
  • then you can execution “hexo s” command Open preview access port.
  • enter “http://localhost:4000“ in you browser .you can watch you blog,but this is in you local computer.

next we will learn how to pull you blog in you github.

remote version

when you have a github account

  • you should build a repository named “you_github name”.github.io.just like me qihao123.github.io
  • in hexo folder search a called _config.yml file and open it(if you cant open it you should better install a notepad++ it can open a lot of format file it include .yml)

  • alter some word in .yml:

    deploy:
    type: git
    repo: https://github.com/"you new repository"
    branch: master
    //note: next":"have a " "
  • then you can execution “hexo d” command remote you blog in github
  • open https://you name.github.io watching you blog!
